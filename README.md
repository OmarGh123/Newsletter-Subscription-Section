# 🌿 Beige Minimalist Sustainable Organic Fashion

A high-fidelity desktop landing page prototype for a premium organic fashion brand. This project focuses on **minimalist UI/UX**, **asymmetric layouts**, and **earth-tone aesthetics** to reflect a sustainable brand identity.

---

## 📖 Project Overview

This landing page serves as the digital storefront for "Really Great Site," a conceptual brand dedicated to eco-friendly apparel. The design emphasizes tranquility and nature, utilizing a neutral beige palette and organic imagery to engage high-end, environmentally-conscious consumers.

### Key Design Elements:
* **Minimalist Navigation:** Clean, spaced-out links for a frictionless browsing experience.
* **Asymmetric Typography:** The background "FASHION" watermark provides depth and artistic flair.
* **Nature-Centric Hero:** A high-definition botanical element (leaves) acts as the visual anchor.
* **Call to Action (CTA):** A high-contrast, deep forest green button to drive conversions.

---

## 🎨 Visual Identity

| Element | Specification |
| :--- | :--- |
| **Primary Palette** | Off-White (#F9F9F7), Forest Green (#1B2607), Soft Gray (#D1D1D1) |
| **Typography** | Elegant Serif (Headings), Clean Sans-Serif (Body/Nav) |
| **Imagery** | Macro organic photography with soft drop shadows |
| **Layout** | 12-Column Grid with wide margins |

---

## 📂 File Structure

* `assets/images/`: Contains the high-res botanical branch and background textures.
* `assets/icons/`: SVG icons for cart, search, and social media.
* `starter-code/front-end/`: The development environment containing:
    * `index.html`: Semantic HTML5 structure.
    * `style.css`: Custom CSS using Flexbox/Grid for the asymmetric layout.

---

## 🛠️ Technical Implementation Goals

1.  **Layered Depth:** Use `z-index` to layer the botanical image over the background watermark text.
2.  **Responsiveness:** Ensure the large "FASHION" text scales appropriately without breaking the layout on smaller screens.
3.  **Hover States:** Implement soft transitions on the "SHOP NOW" button and navigation links.

---

## 🚀 How to Run
1. Clone the repository.
2. Open `starter-code/front-end/index.html` in any modern browser.
3. (Optional) Use **Live Server** in VS Code for real-time styling updates.
